So after understanding what we are going to archive lts get started by setting up the rasberry pies.
A quick summary of what has ty be done is the following:

- Installing an operating system on each of the sub sticks.
- Booting each rasberry pi from a stick.
- Configuring and installing k3s on each rasberry pi.
- Setup the master
- Setup each node.
- Setting up the harddrive for the cluster
